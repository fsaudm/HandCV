# Experiment 5 — Hand Boxes

Create and open floating boxes using hand gestures.

## Gestures

### Create a box (L + L)
1. Make an **L-shape** with each hand — thumb out, index up, other fingers curled
2. A yellow preview rectangle appears between your hands
3. Move hands apart to resize
4. Release the L to drop the box — it opens with text inside

### Drag from shelf
1. **Pinch** (thumb + index) on one of the colored slots at the top
2. Drag it down into the canvas
3. Release pinch to drop

### Open a box
- **Pinch** inside a closed box to open it and reveal its text

## Run

```bash
uv run python experiments/hand_boxes/run.py
```

## Controls

| Key   | Action          |
|-------|-----------------|
| `q`   | Quit            |
| `ESC` | Quit            |
| `c`   | Clear all boxes |

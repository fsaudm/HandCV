- [HandCV](https://github.com/fsaudm/handcv)
  Hand-gesture interactive resume powered by MediaPipe (this app 🌚). Big inspiration from the.poet.engineer and the real-time generative art community (think touchDesigner).
- [DSRS Knowledeg Base](https://github.com/GiesDSRS/knowledge-base/)
  An RAG-less knowledge base for any AI agents to query DSRS' knowledge via Anthropic's Skills. Easily extensible to other data sources, one single unified API. Leverages SKILLS & TOON (Token-Oriented Object Notation) for context window preservation and token efficiency.
- [LLM Containers](https://github.com/GiesDSRS/LLM-containers/)
  Containerized infrastructure for AI model serving using DSRS Kubernetes cluster. vLLM & llama.cpp for LLM, VLM, embedding and guardrails models, among others on A100/H100s with kubernetes and helm. Supports (Q)LoRA, quantizations: GGUF, bitsandbytes, GPTQ; and OpenAI & Anthropic compatible endpoints. 


, and more at [github.com/fsaudm/](https://github.com/fsaudm/) and [github.com/GiesDSRS/](https://github.com/GiesDSRS/)
